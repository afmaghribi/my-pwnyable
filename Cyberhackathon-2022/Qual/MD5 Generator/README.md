# MD5 Generator
Soal code injection pada bahasa C

## Deskripsi
Disediakan layanan untuk generate md5 digest dari inputan yang diberikan pengguna

layanan bisa diakses dengan cara

```
nc HOST PORT
```

## Hint
1. Code Injection
